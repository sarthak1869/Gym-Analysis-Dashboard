# Gym-Analysis-Dashboard
This project aims to provide a comprehensive view of gym workout routines and nutritional data to help users tailor their fitness goals more effectively. It combines exercise data and dietary insights into a single analytical dashboard, making it easier to plan workouts and track nutritional intake.
# Overview
This project aims to provide a comprehensive view of gym workout routines and nutritional data to help users tailor their fitness goals more effectively. It combines exercise data and dietary insights into a single analytical dashboard, making it easier to plan workouts and track nutritional intake.

# Tools & Technologies Used
Power BI – For building dynamic, interactive dashboards

Python – Data cleaning and preprocessing

Pandas & NumPy – For data manipulation and transformation

MS Excel – Used for organizing and storing intermediate datasets

Kaggle API – For sourcing exercise and nutrition data

# Features
# Exercise Data:

Categorized by muscle group, difficulty level, and exercise type

Embedded YouTube links for video references

# Nutrition Data:

Macronutrients tracked: Calories, Proteins, Fats, Carbohydrates, and Fiber

Suitable for diet planning and calorie tracking

# Workflow
Data Collection: Datasets sourced via Kaggle API

Data Cleaning: Handled missing values, standardized units, and removed duplicates using Pandas

Data Analysis: Grouped and aggregated based on workout and nutrition attributes

Visualization: Developed in Power BI for rich data interaction
